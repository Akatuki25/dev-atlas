# manji-standard-server-python (FastAPI + SQLAlchemy)

manji-standard-server の Python スタック。**同一 proto 契約**(`proto/user/v1/user.proto`)から、Go/TS と
同じ proto-first codegen + DDD レイヤを Python に移植する。Go(`../manji-standard-server-go`)が基準実装。

## レイヤ配置（Go/TS と同型）
```
app/
├─ domain/
│  ├─ entity/          *.gen.py   @entity → SQLAlchemy model
│  └─ repository/      *.gen.py   Protocol(interface)
│     └─ mock/         *.gen.py   単体テスト用 mock
├─ dto/               *.gen.py   pydantic(出力境界)
├─ infra/repository/  *.gen.py   SQLAlchemy 実装
├─ usecase/           interface=*.gen.py / 実装=手書き(session/tx)
├─ handler/           *.gen.py   @http → FastAPI router
├─ di/                *.gen.py   wiring(Depends)
└─ main.py            FastAPI app
migrations/           alembic(生成 up)
tools/mss-protoc-gen/ ★ Go の cmd/mss-protoc-gen を Python に移植した生成器
```

## 生成器の移植計画（Go → Python）
Go の `cmd/mss-protoc-gen/generator/*/output/**.tpl`(text/template)を Python テンプレ(Jinja2)に対応させる。
proto descriptor + `@annotation` を読み、以下を生成:

| Go template | Python 生成物 | 実装 |
|---|---|---|
| entity / entity_registry | `app/domain/entity/*.gen.py` | SQLAlchemy declarative model + registry |
| dto | `app/dto/*.gen.py` | pydantic BaseModel（`@timestamp`→`<name>_unix:int`） |
| repository | `app/domain/repository/*.gen.py` | `typing.Protocol` |
| infra_postgres_repository | `app/infra/repository/*.gen.py` | SQLAlchemy 実装 |
| mock | `app/domain/repository/mock/*.gen.py` | in-memory mock |
| usecase | `app/usecase/*.gen.py`(if) + 手書き実装 | session/tx 境界 |
| handler_rest | `app/handler/*.gen.py` | FastAPI `APIRouter`（`@http METHOD /path`） |
| di | `app/di/*.gen.py` | `Depends` wiring |
| (migration-gen) | `migrations/` | alembic up（Entity+snapshot 差分） |

**入力の取り方**: `buf build -o descriptor.binpb` で FileDescriptorSet を出し、Python(`google.protobuf`)で読む。
`@annotation` は leading comment(`SourceCodeInfo`)から抽出（Go 実装と同じ方式）。

## 規約（Go/TS と共通）
- `*.gen.py` は手編集禁止。tx 境界は usecase 層(SQLAlchemy session)。
- test: service/usecase=生成mock(単体, pytest) / handler→DB=実PG(testcontainers, 結合)。`clock` DIで時刻固定。
- uv で依存管理。REST は `@http` から FastAPI router を生成。

### 生成 (protoc のみ)
```bash
python3 tools/mss-protoc-gen/main.py   # proto → app/**/*.py を生成
```
生成ファイルはヘッダ `# Code generated by mss-protoc-gen. DO NOT EDIT.` で識別(Python はドット名を import 不可のため
`*.gen.py` でなく plain `.py` + ヘッダ方式。protobuf の `*_pb2.py` と同流儀)。

### 残タスク
- ⏳ ネストparam の bulk(BulkCreate/BulkUpsert = `repeated CreateUserParams`)の生成(初回除外)。
- ⏳ 実 Postgres 結合テスト(`pytest -m integration`、testcontainers or docker compose)。
- ⏳ migration 生成(mss-migration-gen の Python 版: entity + snapshot 差分 → SQL)。
